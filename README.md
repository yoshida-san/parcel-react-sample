# parcel-react-sample

react を学習する時の最小構成（学習コストが低い）のセット。sass や scss を使わずに postcss+css modules で十分な気がしてる。

- TypeScript
- Parcel
- TSLint + Prettir
- react-router
- unstated-next
- css modules
- postcss

## install

```
~$ npm install
```

## run

```
~$ npm run start
```

access to `http://localhost:1234`
